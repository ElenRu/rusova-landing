# Лендинг Русовой Елены Леонидовны
## Проект: Сайт врача интегративной медицины

---

## Что это за проект

Личный сайт-лендинг **Русовой Елены Леонидовны** — врача интегративной медицины, гинеколога-эндокринолога. Сайт помогает женщинам 45+ найти врача, понять свои симптомы и записаться на консультацию. Основной инструмент привлечения пациентов из Telegram-канала и органического поиска.

---

## Для кого

**Целевая аудитория:** женщины 45–60 лет с симптомами гормонального дисбаланса — приливы, бессонница, усталость, лишний вес, перепады настроения.

**Формат работы:** онлайн-консультации (основной) + офлайн в [ГОРОД].

## Страницы и файлы

### Готово
| Файл | Описание |
|------|----------|
| `index.html` | Главный лендинг — одностраничник со всеми секциями |

### Секции внутри `index.html`
1. **Hero** — заголовок, социальное доказательство, CTA
2. **Pain Points** — «Узнаёшь себя?» — 3 карточки с симптомами
3. **Quiz** — лид-магнит: 5 вопросов → карта симптомов + форма захвата контакта
4. **About** — о враче, дифференциаторы, регалии
5. **How I Work** — 4 шага работы
6. **Reviews** — 3 отзыва пациенток + статистика
7. **Program** — «Гормональный перезапуск 45+», состав и цена
8. **FAQ** — 6 вопросов, CSS-аккордеон
9. **Contact** — финальная форма записи + контакты

### Планируется
- [ ] **Блог / статьи** — отдельные страницы с экспертным контентом (SEO + прогрев из Telegram)
- [ ] **Страница перехода из Telegram** — лёгкий лендинг-мост между каналом и основным сайтом
- [ ] **Политика конфиденциальности** — обязательна для форм с персональными данными
- [ ] **Страница «Спасибо»** — после записи / оплаты

## Дизайн-система

**Стиль:** Flat / Clean — без теней, без градиентов. Медицинский, но тёплый.

| Токен | Значение | Применение |
|-------|----------|------------|
| `--primary` | `#2D5F4E` | Тёмно-зелёный — основной цвет, кнопки, акценты |
| `--primary-light` | `#EAF4EF` | Бледно-зелёный — фоны секций |
| `--accent` | `#C4805A` | Тёплая терракота — финальный CTA, цена |
| `--text` | `#1A1A1A` | Основной текст |
| `--text-muted` | `#6B6B6B` | Второстепенный текст |
| `--bg` | `#FAFAF8` | Фон страницы |
| `--white` | `#FFFFFF` | Карточки, формы |

**Типографика:** `system-ui, -apple-system, sans-serif` — системный шрифт, без внешних зависимостей

**Радиусы:** 12px — карточки, 8px — кнопки

**Брейкпоинт:** 768px — мобильная версия

---

## Технический стек

- **Чистый HTML + CSS + Vanilla JS** — ноль зависимостей, открывается напрямую в браузере
- **Telegram Bot API** — уведомления о заявках (настроить `BOT_TOKEN` и `CHAT_ID`)
- Нет npm, нет фреймворков, нет сборщиков

| Плейсхолдер | Что подставить |
|-------------|----------------|
